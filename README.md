# Brain Games

[![Actions Status](https://github.com/Denisido/backend-project-44/actions/workflows/hexlet-check.yml/badge.svg)](https://github.com/Denisido/backend-project-44/actions)

Brain Games is a set of console mini-games.

The player has to answer three questions correctly.
If the player gives a wrong answer, the game ends and shows the correct answer.

## Requirements

* Node.js 18 or higher
* npm

## Installation

```bash
git clone https://github.com/Denisido/backend-project-44.git
cd backend-project-44
make install
npm link
```

## Usage

You can run the main greeting command:

```bash
brain-games
```

You can also run any game directly.

### Brain-even

Answer `yes` if the number is even, otherwise answer `no`.

```bash
brain-even
```

### Brain-calc

Calculate the result of the expression.

```bash
brain-calc
```

### Brain-gcd

Find the greatest common divisor of two numbers.

```bash
brain-gcd
```

### Brain-progression

Find the missing number in the arithmetic progression.

```bash
brain-progression
```

### Brain-prime

Answer `yes` if the number is prime, otherwise answer `no`.

```bash
brain-prime
```
